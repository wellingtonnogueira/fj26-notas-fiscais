Projeto fj26-notas-fiscais

Curso FJ-26 ministrado pelo instrutor Alexandre Gama (@alexandregama) na Caelum.

A base deste projeto (curso) consiste na geração da interface com usuário utilizando JSF. A camada de persistência já fora concebida previamente pois não é escopo deste curso porém recebendo melhorias no decorrer do curso.

Aproveitando os conhecimentos adquiridos no curso FJ-25 será adicionada algumas features a este projeto visando melhorar alguns pontos (além de praticar conhecimentos adquiridos).
